# F1 Car Project

A visually rich **Formula 1 themed website project** built using HTML and CSS. This project presents an immersive F1 experience with multiple pages such as home, about, specs, contact, and race-track related content, designed with a modern motorsport-inspired interface.

## Overview

This project is a static front-end website created for a college project. It focuses on Formula 1 cars, their specifications, racing atmosphere, and premium UI design. The interface uses dark tones, red highlights, and smooth layouts to match the speed and intensity of F1.

## Features

- Multi-page F1 themed website
- Modern and premium dark UI
- Dedicated pages for car specs, contact, and race tracks
- Responsive layout for different screen sizes
- Hover effects and interactive navigation styling
- Motorsport-inspired visual theme
- Clean section-based design for easy browsing

## Pages Included

- **Home** – Main landing page introducing the F1 project
- **About** – Information about Formula 1 and the project concept
- **Specs** – Technical specifications and details of F1 cars
- **Tracks** – A dedicated page for famous Formula 1 race tracks
- **Contact** – Premium-styled contact page for user interaction

## Technologies Used

- HTML5
- CSS3
- Tailwind CSS (if included in your project)
- JavaScript (if used for interactions)

## Design Style

The website is designed to create a premium and sporty feel inspired by Formula 1 branding. Key design choices include:

- Dark background theme
- Red highlight color for emphasis
- Card-based layouts
- Premium glass/3D-style sections
- Full-screen hero sections
- Modern navigation bar styling
